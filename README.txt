PNG decode add-in for D-Pixed
Version 1.01


0. はじめに

    このソフトウェアはD-Pixed用の，画像デコーダです。
    PNGファイルをロードできるようになります。
    別途PNGファイルがロードできるSusieプラグインが必要です。

2. インストール方法

    D-Pixedをインストールしたディレクトリに「dpdecpng.dll」と「ifpng.spi」を置
    いて下さい。「ifpng.spi」はPNGをロードするSusieプラグインです。PNGのプラグ
    インは色々な人が作っているので別の名前かもしれませんが、その場合も
   「ifpng.spi」という名前で置いて下さい。
